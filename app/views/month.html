1. Introduction

Applicants are only counted once. If an applicant has applied to more than one course, they’re counted under the highest row (in the table) which applies.

For example, if a candidate has received one offer, declined one offer, and is awaiting a provider decision on a third course, they're counted under “Received an offer”.

People can apply for up to 3 courses in their initial application. If these are all unsuccessful, the applicant can apply again an unlimited number of times, but for only a single course at a time.

Initial application – this shows the number of candidates on their first application (candidates can apply for up to 3 courses in this application).

Apply again – this shows the number of applicants who have applied again after an unsuccessful first application. They can “apply again” as many times as they like, but can only apply to one course at a time.
